mysql5.6使用老格式密码

三 23 三月 2016

Filed under 运维

Tags mysql

转发的，找不到原文了。。

root@mysql-5.1.51> SET SESSION old_passwords=0; Query OK, 0 rows affected (0.00 sec)

root@mysql-5.1.51> SELECT PASSWORD('123456'); +-------------------------------------------+ | PASSWORD('123456') | +-------------------------------------------+ | *6BB4837EB74329105EE4568DDA7DC67ED2CA2AD9 | +-------------------------------------------+ 1 row in set (0.00 sec)

root@mysql-5.1.51> SET SESSION old_passwords=1; Query OK, 0 rows affected (0.00 sec)

root@mysql-5.1.51> SELECT PASSWORD('123456'); +--------------------+ | PASSWORD('123456') | +--------------------+ | 565491d704013245 | +--------------------+ 1 row in set (0.00 sec

实际操作： 登陆mysql，使用SET SESSION old_passwords=0; //设置老格式密码 SET PASSWORD FOR 'webuser'@'localhost' = PASSWORD('123456') //然后为用户设置密码。

修改配置文件： [client] secure_auth = 0 [mysqld] old_passwords=1

成功登陆！

忘记用户密码时： [mysqld] skip-grant-tables

然后不用口令进入mysql修改密码：

UPDATE mysql.user SET Password=PASSWORD("newpassword") WHERE User='root' and Host='localhost';
