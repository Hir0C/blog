Nginx + Django 部署

二 05 七月 2016

Filed under 运维

Tags nginx django

配置

nginx配置

server {
    listen      80;
    server_name localhost
    charset     utf-8;

    client_max_body_size 75M;

    location /static {
        alias /home/django_site/static; 
        mp4; #项目中的视频文件存在这个目录里
        proxy_ignore_headers X-Accel-Expires Expires Cache-Control Set-Cookie;
    }

    location / {
        include     uwsgi_params;
        uwsgi_pass  unix:///usr/local/nginx/uwsgi.sock;
    }
}

uwsgi配置

[uwsgi]
socket = /usr/local/nginx/uwsgi.sock
chdir = /home/django_site/
wsgi-file = django_site/wsgi.py

processes = 2
threads = 4

chmod-socket = 664
chown-socket = www:www

django配置

关闭DEBUG。别忘了设置允许所有主机访问，否则会报错400 

DEBUG = False
ALLOWED_HOSTS = ['*']

运行

启动uwsgi

uwsgi --ini ./conf.ini > /dev/null 2>&1 &

每次更新py文件后，需要重启uwsgi
