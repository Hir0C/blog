Filed under 运维

Tags mail

因为我们使用了mail http api 接口发邮件,如果同时爆出大量发邮件请求,讲会堆积大量邮件队列,造成服务器卡死,临时解决方案就是不断循环检测邮件队列数量进行清理。

转贴

Postfix中有一套Mail Queue Management机制,所有队列中的邮件都可以全自动的处理,但在发送大量邮件的时候,有必要对这个队列进行手工的维护处理,比如说,删除队列中的邮件. 以下是一些常用的命令：

列出目前在 Mail Queue 中的邮件 

mailq

刪除所有在 Queue 中的邮件 

postsuper -d ALL

刪除所有正在 deferred 队列中的邮件 (删除曾经发送失败的邮件 ) 

postsuper -d ALL deferred

刪除所有正在 deferred 队列中的郵件 ( 可看出哪些信被刪除了 )

python使用中遇到的一些问题

四 24 三月 2016

Filed under 日常

Tags python

常用特殊符号

左位移 <<

转换二进制后去掉最左边y位

计算公式 x * (2 ** y)

比如 4 << 3 等于 4 * ( 2 ** 3) 等于 32

右位移 >>

转换二进制后去掉最右边y位

计算公式 x / (2 ** y)

比如 32 >> 3 等于 32 / ( 2 ** 3) 等于 4

右位移实际应用：

右位移10位，计算数据占用磁盘空间k(比如1100000 >> 10 = 1074)

再右位移10位，计算数据占用磁盘空间M(1074 >> 10 = 1)

按位与 &

转换二进制后比较，只有双方都唯一时才得1

用来计算奇偶数

比如 4 &1 = 0 ...
