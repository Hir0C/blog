windows10 清空DNS缓存

五 25 三月 2016

Filed under 日常

Tags windows

转发

於桌面按住 Windows + X 按鍵。
選擇 Command Prompt (以管理員執行)。
在彈跳視窗中輸入 ipconfig /release
等待數秒回報此 IP 位址已被釋放。
在彈跳視窗中輸入 ipconfig /renew
等待數秒回報此 IP 位址已被重建。
在彈跳視窗中輸入 ipconfig /flushdns
關閉視窗並再次嘗試連線。
